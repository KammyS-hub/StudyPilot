# StudyPilot AI

StudyPilot AI is an AI-powered academic productivity assistant built using Python.

## Features

- Study Planning
- DSA Progress Tracking
- Performance Analytics
- Personalized Recommendations
- Skill Gap Analysis

## Technologies Used

- Python
- NLP
- Data Analysis

## Project Structure

- app.py → Main application
- planner.py → Study planning module
- dsa_tracker.py → DSA tracking module
- analytics.py → Performance analytics
- core/ → NLP and recommendation modules

## Future Improvements

- Streamlit UI
- Resume-based learning recommendations
- Dashboard visualizations
